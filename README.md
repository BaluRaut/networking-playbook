# Networking Pro Playbook

**26 topics** on the wire under everything you deploy — IP, TCP/UDP, DNS, TLS, HTTP, proxies, firewalls, and a debug-in-layers runbook. Each topic with **real commands, real output, a plain-English analogy, and the gotcha that pages you.**

🔗 **Live:** https://baluraut.github.io/networking-playbook/

## What's inside

**Foundations (01–08):** the layer model, IP & CIDR, MAC & ARP, routing & gateways, TCP (the handshake), UDP (fire & forget), ports & sockets (ss), NAT.

**Names, Web & Load (09–16):** DNS (dig), resolv.conf & hosts, HTTP anatomy, TLS & certificates, load balancing L4/L7, reverse proxies (nginx), ping & ICMP, traceroute & mtr.

**Debugging & Ops (17–26):** curl mastery, tcpdump, TIME_WAIT & TCP states, iptables & nftables, ssh mastery & tunnels, ip addr & interfaces, the "is it down?" runbook, latency vs bandwidth, cloud networking mapped, and a **capstone: one request, end to end.**

Includes a **round-based quiz** that **saves progress in the browser**.
